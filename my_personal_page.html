About Me

Rica Mhey A. Malangen

I’m a 2nd year IT student at Panpacific University, currently working as a layout artist for our school publication, THE TRUTH. I joined the team in August to gain hands-on experience and improve my editing and layout design skills. Despite having no prior background in layout work during high school, I challenged myself to learn and adapt.

Through consistent practice and collaboration, I’ve grown confident in visual arrangement and editorial design. One of my proudest milestones was helping complete Volume 4 of our news article—a project that reflects both my technical progress and creative discipline.

Beyond publication work, I’ve built a range of academic projects that reflect my growth in both technical and creative domains. These include a Student Record Management System in C++, which handles student data through structured programming and file operations; an MP3 Player UI prototype in Figma, designed with intuitive controls and modern styling for a Human-Computer Interaction (HCI) project; and Abviente, an advocacy site built in Python that promotes tree planting through interactive content and clean layout. I also developed RicaInstax, an Instagram-inspired social media prototype in Figma that focuses on personalized content sharing and component reuse; a Hotel Management System in Python that streamlines room bookings, guest check-ins, and billing through a centralized interface; and RicaVerse, my personal portfolio page crafted in HTML and CSS to showcase my identity, skills, and featured work. These experiences have strengthened my foundation in coding, graphic design, and digital prototyping—allowing me to blend structure with creativity across different platforms.

Featured Work

MP3 Player

A sleek and interactive MP3 player UI prototype created in Figma, focused on intuitive user experience and modern visual styling. The design includes essential playback controls, playlist navigation, color adjustment, and album art integration. Built for a Human-Computer Interaction (HCI) project, this layout emphasizes clarity, accessibility, and responsive design principles, showcasing practical skills in digital prototyping, layer organization, and component reuse.

Figma Prototype

Student Record Management System

A basic yet functional C++ console application developed using Programiz, designed to manage student records efficiently. The system allows users to add, view, search, update, and delete student data. Built with structured programming and clear logic flow, this project demonstrates practical understanding of file handling, functions, and data structures in C++.

C++ Programiz

Advocacy Site

Abviente is a purpose-driven advocacy website promoting tree planting and environmental awareness. It was conceptualized and built during our Human-Computer Interaction Lab Exam, showcasing our skills in user-centered design, digital prototyping, and sustainability-focused content.

Figma Prototype Website

Instagram Prototype

RicaInstax is a conceptual prototype of a social media application inspired by Instagram, tailored for personalized content sharing. Built in Figma as part of our Human-Computer Interaction Lab, it highlights user interface design, component reuse, and interactive layout principles.

Figma Prototype

Hotel Management System

A Hotel Management System (HMS) is a software application designed to streamline and automate the day-to-day operations of a hotel, such as room bookings, guest check-ins and check-outs, payments, and room management. It helps hotel staff manage reservations, track room availability, store guest details, and handle billing efficiently through a centralized platform.
